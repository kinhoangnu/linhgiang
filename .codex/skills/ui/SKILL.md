# UI

Use this workflow for visual and interaction work.

## Visual System

- Quiet, practical household operations UI.
- Mobile-first, dense enough for repeat use, with clear touch targets.
- Cards are for repeated chores, shopping items, modals, or framed tools only.
- Keep colors balanced: neutral surfaces with green, blue, and warm price accents.
- Avoid decorative clutter, marketing hero layouts, and nested cards.

## Workflow

1. Read the relevant component and nearby `css/styles.css` section.
2. Preserve CSS organization and responsive constraints.
3. Check widths around 375px, 768px, and 1280px for significant UI work.
4. Keep touch targets at least 44px.
5. Run `npm run build`.
6. Use browser or screenshot checks for meaningful visual changes when feasible.
